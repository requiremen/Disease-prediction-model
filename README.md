# 📈 DSE Index Prediction Model

A Machine Learning based prediction model designed to forecast trends of the **Dhaka Stock Exchange (DSE)** index using historical market data and technical indicators.

This project explores time-series forecasting techniques to predict future market movements and assist in data-driven investment decisions.

---

## 🚀 Project Overview

The goal of this project is to:

* Collect and preprocess historical DSE index data
* Perform exploratory data analysis (EDA)
* Engineer meaningful financial features
* Train machine learning models for prediction
* Evaluate model performance using standard metrics

---

## 🧠 Models Used

* Linear Regression
* Random Forest Regressor
* XGBoost Regressor
* LSTM (Long Short-Term Memory) Neural Network

You can modify or add additional models depending on your research needs.

---

## 📊 Features Used

* Open Price
* High Price
* Low Price
* Close Price
* Volume
* Moving Averages (MA)
* RSI (Relative Strength Index)
* MACD (Moving Average Convergence Divergence)
* Daily Returns

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/DSE-Prediction-Model.git
cd DSE-Prediction-Model
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the training script:

```bash
python train.py
```

Run prediction:

```bash
python predict.py
```
